Spotify Streaming Trends Analysis

An end-to-end data analytics project leveraging Spotify Developer API, Python, SQL, and data visualization to uncover music streaming insights.

Project Overview

Problem Statement

Spotify generates massive streaming data daily. The challenge is to convert this raw data into structured insights that reveal artist popularity, genre dominance, regional listening behavior, and trends over time.

What I Did

  • Collected datasets from Kaggle and Spotify Developer API
  • Cleaned and transformed data using Python (Pandas)
  • Designed SQL schema and loaded cleaned data
  • Executed analytical SQL queries for insights
  • Built visualizations in Jupyter Notebook

Tools & Technologies

Python
Spotify API
SQL
Pandas
Matplotlib
Jupyter Notebook

Sample SQL Query

SELECT artist_name,
       SUM(streams) AS total_streams
FROM spotify_streams
GROUP BY artist_name
ORDER BY total_streams DESC
LIMIT 10;
